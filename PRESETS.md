# WORLDVIEW // Keyboard Shortcuts & Demo Flows

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `1` | Standard visual mode |
| `2` | CRT mode (scanlines, chromatic aberration) |
| `3` | Night Vision mode (green monochrome, noise) |
| `4` | FLIR/Thermal mode (heat gradient) |
| `5` | Pixelated/Sat-recon mode |
| `Q` | Jump to POI #1 in current city |
| `W` | Jump to POI #2 in current city |
| `E` | Jump to POI #3 in current city |
| `R` | Jump to POI #4 in current city |
| `T` | Jump to POI #5 in current city |
| `Tab` | Toggle shader controls panel |
| `Esc` | Clear entity tracking |
| `/` | Focus search bar |

## Demo Flows

### Flow 1: Pentagon Surveillance
1. Select "Washington DC" from the city dropdown
2. Press `Q` → fly to The Pentagon
3. Toggle "MIL FLIGHTS" layer ON
4. Toggle "SATELLITES" layer ON
5. Press `4` → switch to FLIR/Thermal mode
6. Open shader panel (Tab), adjust intensity and edge enhancement
7. Click a military aircraft to track it

### Flow 2: Global Seismic Monitoring
1. Toggle "SEISMIC" layer ON
2. Press `3` → Night Vision mode
3. Zoom out to see the full globe
4. Observe earthquake markers pulsing by magnitude
5. Select different cities to explore seismic hotspots

### Flow 3: New York City Intelligence Sweep
1. Select "New York" from the city dropdown
2. Press `Q` → Empire State Building
3. Toggle "CIV FLIGHTS" layer ON
4. Toggle "SATELLITES" layer ON
5. Press `2` → CRT mode
6. Press `W` → Statue of Liberty
7. Press `E` → One World Trade Center
8. Switch detection mode to "FULL" for all entity labels

### Flow 4: Tokyo Night Ops
1. Select "Tokyo" from the city dropdown
2. Press `Q` → Tokyo Tower
3. Press `3` → Night Vision mode
4. Toggle "CIV FLIGHTS" ON
5. Adjust noise and bloom sliders in shader panel
6. Cycle through POIs: `W` (Skytree), `E` (Shibuya), `R` (Imperial Palace)

### Flow 5: Dubai Thermal Recon
1. Select "Dubai" from the city dropdown
2. Press `Q` → Burj Khalifa
3. Press `4` → FLIR/Thermal mode
4. Toggle "SATELLITES" ON
5. Filter satellites to GEO only
6. Press `W` → Palm Jumeirah (zoom out to see the island)

### Flow 6: Austin CCTV Network
1. Select any city, then toggle "CCTV FEEDS" layer ON
2. The camera will show Austin, TX traffic cameras
3. Zoom into Austin area to see camera positions
4. Camera feeds refresh automatically every 5 seconds

### Flow 7: Live Traffic Analysis
1. Fly to any urban area
2. Toggle "TRAFFIC SIM" layer ON
3. Watch as roads load sequentially: highways → arterials → local
4. Press `5` → Pixelated/Sat-recon mode for satellite imagery look
5. Adjust pixelation slider for the desired reconnaissance aesthetic
